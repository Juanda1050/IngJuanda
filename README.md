# IngJuanda Portfolio

Portafolio profesional con estética Visual Studio Code dentro de una ventana estilo macOS.

## Stack

- React + TypeScript + Vite
- TailwindCSS + componentes estilo shadcn/ui
- Zustand
- react-i18next (es/en)
- Framer Motion
- Lucide React

## Scripts

- `npm run dev`
- `npm run lint`
- `npm run build`

## Estructura

- `src/app`
- `src/components`
- `src/features`
- `src/hooks`
- `src/i18n`
- `src/layouts`
- `src/lib`
- `src/shared`
- `src/store`
- `src/types`
